# ClearChat

ClearChat is a real-time group chat web application with built-in accessibility enhancements.  
Developed for the Advanced Software Engineering course at Algoma University.

## Features
- Real-time messaging (React + Socket.IO)
- Username-based chat entry
- Accessibility Control Panel
  - Text size, line & letter spacing
  - High contrast mode
  - Reduce motion
  - Text-to-Speech
- Persistent user settings (localStorage)

## Tech Stack
- React
- Node.js
- Socket.IO
- Tailwind CSS
- Web Speech API

Academic project – Incremental & Agile development.
